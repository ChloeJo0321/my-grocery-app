# Grocery App
A simple grocery app built with React and Node.js.

## 🛠 Tech Stack
- React
- Node.js
- Express
- MySQL

## 🚀 How to Run
1. Clone the repository:
```bash
git clone https://github.com/username/my-grocery-app.git
cd my-grocery-app
```
2. Install dependencies:
```bash
npm install
```
3. Start the server:
```bash
npm start
```
4. Start the frontend
```bash
npm run dev
```
